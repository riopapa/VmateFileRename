vmate file rename/copy (2020-10-10)

Function : Convert vmate files into valid date time format

o Current Snoppa vmate app creates media file including colon(:) and timezone is not local.
o Some video play app in android phone can NOT understand this colon, and spit out that video
o This app will rename or copy vmate media files into Camera Folder (DCIM/vmate) with proper name.

How to operate

o When this app is loaded, it will display all the file list you downloaded from vmate gimbal.

o On Setting screen,
You may set time zone to fit your local time
You may select rename or copy option
If copy with delete option is selected, each files in source folder is copied into destination folder, it will be deleted, like moving..
You know that original files are still in vmate gimbal sdcard.
If ready, click play icon, then copying will be started while displaying current progress.
     from : \Vmate\sd/DCIM/100HSCAM
     into : DCIM\vmate

Notice

o No guarantee for improving functions

o This app will disparate when snoppa vmate gives proper file name.
